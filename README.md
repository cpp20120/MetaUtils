# Useful Metaprogramming library

[![CMake](https://img.shields.io/badge/CMake-3.26+-blue.svg)](https://cmake.org/)
[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)
[![](https://tokei.rs/b1/github/cpp20120/MetaUtils)](https://github.com/cpp20120/MetaUtils).

## Contains

- concepts
- traits
- static reflection
- metafunctions
- smart_cast
- type_builder
- views
- constants
- dynamic_optional

## At this moment not ready for production use

I need to create a unified API and add more features to certain parts. Additionally, I do not have any tests at this time (but I will be conducting them in the future).
